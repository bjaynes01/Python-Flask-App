# Flask Web App Tutorial

## Setup & Installation

Make sure you have the latest version of Python installed.

```bash
git clone <repo-url>
```

```bash
pip install -r requirements.txt
```

## Running The App

```bash
cd hello_flask

source bin/activate

pip install flask

pip install flask-login

pip install flask-sqlalchemy

python main.py
```



## Viewing The App

Go to `http://127.0.0.1:5000`
